## 업로드 방법

1. 로컬 폴더 열기: `C:\Users\user\Documents\Codex\2026-05-27\4\deliverables\triom_github_upload_batches`
2. `batch_01` 폴더를 열고, 그 안의 파일과 폴더 전체를 GitHub 업로드 화면에 드래그
3. Commit message: `Upload TriOm batch 01`
4. `Commit changes` 클릭
5. `batch_02`부터 `batch_10`까지 반복
6. 마지막 배치 후 `Actions` 탭에서 `Deploy TriOm to GitHub Pages` 완료 확인

## 주의

- `batch_XX` 폴더 자체를 올리지 않음
- 반드시 각 `batch_XX` 폴더 안의 내용만 선택
